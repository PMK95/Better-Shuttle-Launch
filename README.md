# Better Shuttle Launch

RimWorld 1.6 Odyssey QoL mod.

## Build

Double-click `Build.bat` from the repository root.

Manual command:

```powershell
dotnet restore Source/BetterShuttleLaunch/BetterShuttleLaunch.csproj
dotnet build Source/BetterShuttleLaunch/BetterShuttleLaunch.csproj -c Release --no-restore
```

The release DLL is copied to:

```text
Better Shuttle Launch/1.6/Assemblies/BetterShuttleLaunch.dll
```
